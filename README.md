# Netflix Data Analysis

## Problem Statement
Netflix has a vast collection of movies and TV shows across different countries, genres, and ratings.  
The objective of this project is to perform Exploratory Data Analysis (EDA) on the Netflix dataset to understand content distribution, production trends, rating patterns, genre popularity, and yearly growth of content.

This analysis helps derive actionable insights into Netflix’s content strategy and viewer preferences.

## Objectives
- Analyze the distribution of Movies and TV Shows on Netflix
- Identify top countries producing Netflix content
- Analyze content ratings and their trends over time
- Explore genre popularity and trends
- Study yearly growth in Netflix content additions
- Perform data cleaning and handling of missing values

## Dataset Information
- Dataset: Netflix Movies and TV Shows
- Attributes include:
  - Title
  - Type (Movie / TV Show)
  - Director
  - Cast
  - Country
  - Rating
  - Duration
  - Release Year
  - Date Added
  - Genre (Listed In)

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Approach / Methodology
1. Data loading and initial exploration
2. Handling missing values (imputation and filtering)
3. Data cleaning and transformation
4. Feature analysis using groupby and aggregation
5. Exploratory Data Analysis (EDA)
6. Data visualization to identify trends and patterns
7. Insight generation and interpretation

## Key Analysis Performed
- Content type distribution (Movies vs TV Shows)
- Country-wise content production analysis
- Rating distribution and trends
- Genre popularity analysis
- Year-wise content addition trend
- Analysis of directors, cast, and duration patterns

## Key Insights
- The United States produces the highest number of Netflix titles, followed by India
- Movies make up a larger portion of Netflix content compared to TV Shows
- TV-MA and TV-14 are the most common content ratings
- International Movies and Dramas are among the most popular genres
- Netflix content additions increased significantly after 2015

## Conclusion
This project demonstrates effective use of Python for data cleaning, exploratory data analysis, and visualization.  
The insights derived provide a clear understanding of Netflix’s content trends, production patterns, and audience targeting strategy.
